CS101 Personal Project : Blackjack

Basic Terminal Gameplay of BlackJack

How To Play:
1. Type "Start" to the terminal
2. Game will start with your current cards and values
3. Press H to Hit, Press S to Stand
4. After pressing S, dealer will make play. Whowever has the highest value without busting wins
5. Press Y to play again, Press N to end the game
